---------------------------------------------------
1.  Download and install Git v1.7.1 from:

     http://book.git-scm.com/2_installing_git.html

---------------------------------------------------
2. Verify Git is installed by running from the command line:

      git --version

---------------------------------------------------
3.  Make a directory:

     mkdir spring-demos

---------------------------------------------------
4.  Move into that directory:

     cd spring-demos

---------------------------------------------------
5. Gordon's Core Class Demos:

     git clone git://github.com/gordonad/core-spring-demos.git
